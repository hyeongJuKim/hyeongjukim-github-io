if __name__ =='__main__' 은 왜 사용할까?

Sep 12, 2019 9:01·159 words ·1 minutes read

먼저 __name__ 이 무엇인지 부터 알아보자.
__name__ 은 모듈의 이름을 저장하는 파이썬 내장 변수이다.

파일을 실행하거나 import 할 때 함수가 아닌 0 deps의 코드를 실행한다.
직접 실행하게 되면 __name__ 변수에 ‘__main__‘ 값이 할당되고,
import하게 되면 해당 모듈의 이름이 값이 할당된다.

해당 모듈을 import 할 때 자동으로 실행되는 것을 방지 하기 위해 if __name__ == ‘main‘ 을 사용한다.

print_hello.py를 직접 실행해보자.

print_hello.py

def print_hello:
    print('hello')

print('print_hello.py __name__:', __name__)
print('test hello')

실행결과

print_hello.py __name__: __main__
test hello

__name__ 이라는 변수에는 main 이 담겨져 있고
0 deps 코드가 실행이 된다.

다른 파일에서 import해보자.
main.py

import print_hello

print('main.py __name__:', __name__)

실행결과

print_hello.py __name__: hello
test hello
main.py __main__

__name__ 이라는 변수에는 print_hello 가 담겨져 있고,
print_hello.py 를 import 하면서 0 deps 코드를 실행한다.
import 할 때 자동적으로 코드가 실행되는 것을 방지하기 위해 비교하는 조건문을 사용하는 것이다.

if __name__ =='__main__':
    print('직접 실행됨')
else:
    print('import되어 실행됨')
